eDrugLookup

Informasi penggunaan dan penanganan data

eDrugLookup adalah alat bantu pencarian obat untuk workflow resep di eClinic. Hasil pencarian tetap harus diverifikasi oleh tenaga kesehatan sebelum tindakan atau resep disimpan.

Ringkasan penting

Extension ini tidak menggantikan penilaian klinis. Pengguna wajib memverifikasi pasien, nama obat, sediaan, dosis, dan instruksi penggunaan sebelum melanjutkan.

Tujuan extension

Membantu pencarian kandidat obat di area resep, terutama pada konteks Non Racik dan modal Cari Obat, tanpa mengambil alih keputusan akhir dari pengguna.

Batas fungsi

  • Extension membantu lookup dan suggestion, bukan keputusan medis.
  • Extension tidak ditujukan untuk submit resep secara otomatis.
  • Extension tidak dimaksudkan menggantikan verifikasi manual oleh tenaga kesehatan.

Penanganan data

  • Extension dapat membaca data yang tampil pada halaman EMR untuk kebutuhan konteks pencarian.
  • Data dipakai untuk mendukung fungsi operasional extension pada halaman yang sedang dibuka.
  • Penggunaan extension harus mengikuti otorisasi, SOP fasilitas kesehatan, dan kebijakan sistem yang berlaku.
  • Hanya pengguna yang berwenang yang boleh menggunakan extension ini pada data pasien.

Catatan perilaku sistem

  • Extension dirancang untuk tetap native-first pada workflow resep eClinic.
  • Extension tidak men-trigger tombol Tambahkan Obat secara otomatis.
  • Extension tidak men-trigger tombol Simpan secara otomatis.
  • Extension tidak boleh digunakan untuk menghindari proses verifikasi internal fasilitas kesehatan.
Gunakan extension ini secara hati-hati dan sesuai kebijakan internal fasilitas kesehatan.
